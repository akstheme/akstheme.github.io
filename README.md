## Overview

This website is designed to present an academic profile in a clean, modern, and responsive format. It serves as a central hub for research activities, publications, teaching experience, projects, awards, and professional achievements.

The website is completely static and is intended to be hosted using **GitHub Pages**.

## Local Development

Clone the repository

```bash
git clone https://github.com/your-username/portfolio.git
```

Navigate to the project directory

```bash
cd portfolio
```

Open

```
index.html
```

or launch using VS Code Live Server.

---

## Deployment

This project is designed to be deployed using **GitHub Pages**.

1. Push the repository to GitHub.
2. Open **Settings**.
3. Navigate to **Pages**.
4. Select the `main` branch.
5. Save the configuration.
6. Your website will be published automatically.
